Vendor Dashboard

Vendor & Admin Dashboard

The secure vendor dashboard was built in React includes custom signup and login pages, dashboard, and admin area. It allows authenticated food vendors to add and edit events, and edit food stand information. The admin area features the ability to create new food stands, and switch vendors to directly edit their events and information.
